# Hello Health App

## Description

This project is about creating a python environment, Git workflow and repository structure to build a strong foundation befor moving on to more advanced programming tasks. 

The hello health app is an example to show that the environment is working correctly.

## Installation instructions

1. *Clone the repository*

cloning the repository

command: git clone https://github.com/FilipTamang/data1300-oblig-1.git

entering the project directory

command: cd FilipTamang-oblig-1

2. *Create a virtual environment*

creating the virtual environment

command: python -m venv venv

3. *Activate the virtual environment* 

On Windows Powershell:

command: venv\Scripts\Activate.ps1

On macOS/Linux:

command: source venv\bin\activate

4. *Run the application*

command: python hello_health.py

## Requirements

Python 3.10 or higher

No external dependencies for Obligation 1
